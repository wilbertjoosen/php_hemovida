• [RUS]

    Путь до исполняемого файла. Пользователь должен ввести путь до исполняемого файла начиная с имени проекта. Например, пользователь создал проект "Hello_world" и положил в workspace исполняемый файл run.bat. Тогда он должен ввести следующее : Hello_world/workspace/run.bat. Окончательный путь образуется следующим образом HUDSON_HOME/jobs/path Поэтому обязательно нужно определить HUDSON_HOME на машине где хостуется Hudson.

  • [ENG]

    Executable file. The user must enter the path to the executable file from the project name. For example, a user created the "Hello_world" and put it in the workspace executable file run.bat. Then he must enter the following: Hello_world / workspace / run.bat. The final path is formed as follows HUDSON_HOME / jobs / path So make sure to define at HUDSON_HOME.
